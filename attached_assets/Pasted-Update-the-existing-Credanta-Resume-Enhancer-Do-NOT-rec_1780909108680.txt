IMPROVE ANALYSIS ENGINE
────────────────────────

Enhance the current resume analysis to include:

1. Overall Resume Score
Score from 0–100.

2. Category Scores:
- Completeness
- Readability
- Professional Summary
- Action Verbs
- Keyword Match
- Formatting Clarity

3. Missing Section Detection:
Check for:
- Summary / Professional Summary
- Work Experience
- Education
- Certifications
- Licenses
- Skills
- Contact Information

4. Weak Word Detection:
Detect weak phrases and suggest replacements.

Examples:
Responsible for → Managed / Coordinated / Led
Helped → Assisted / Supported / Improved
Worked on → Developed / Managed / Executed
Did → Completed / Performed / Delivered
Participated in → Collaborated / Contributed / Coordinated

5. Action Verb Suggestions:
Suggest stronger verbs based on common resume language.

6. Keyword Analysis:
Compare resume text against the selected target role.

Add role keyword libraries for:
- Travel Nurse
- Registered Nurse
- ICU RN
- Telemetry RN
- Med-Surg RN
- Home Health RN
- Legal Nurse Consultant
- Software Developer
- Teacher
- General Professional

For each role, return:
- Found keywords
- Missing suggested keywords
- Keyword match percentage

7. Formatting Checks:
Detect:
- very long paragraphs
- overly short resume
- missing bullet points
- inconsistent spacing
- repeated phrases
- weak summary
- missing measurable outcomes

8. Improvement Checklist:
Generate clear checklist items:
- Add a professional summary
- Add measurable outcomes
- Add certifications section
- Add specialty keywords
- Use stronger action verbs
- Shorten long paragraphs
