────────────────────────
REWRITE LOGIC
────────────────────────

The local rewrite engine should improve text using:

1. Section Detection
Detect common sections:
- Summary
- Professional Summary
- Experience
- Work Experience
- Education
- Certifications
- Licenses
- Skills
- References

2. Summary Rewriting
If a summary exists:
- rewrite it into each tone version

If no summary exists:
- generate a conservative summary using only facts found in the resume

Example:
"Experienced registered nurse with background in acute care, patient care, documentation, and interdisciplinary collaboration."

Only include specialties if detected in the resume.

3. Bullet Rewriting
Detect bullet lines and improve them.

Weak phrase replacements:
- Responsible for → Managed / Coordinated / Provided
- Helped → Assisted / Supported / Improved
- Worked on → Supported / Completed / Contributed to
- Did → Performed / Completed
- Participated in → Collaborated with / Contributed to
- Took care of → Provided care for
- Charted → Documented
- Gave meds → Administered medications

4. Healthcare-Specific Improvements
If healthcare terms are detected, improve wording with accurate professional language.

Examples:
- patients → patients
- meds → medications
- vitals → vital signs
- charting → clinical documentation
- report → handoff communication
- doctor → provider / physician when appropriate

5. Formatting Cleanup
- remove excessive blank lines
- standardize bullet spacing
- preserve section headings
- keep readable line breaks
- keep original order of sections
